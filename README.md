## 关于采矿工程

采矿工程是一门旨在**安全、高效、经济且环保**地从地壳中开采矿产资源的工程技术学科。
根据开采方式的不同，一般分为：
- **露天开采**
- **地下开采（井工开采）**
本项目主要针对**固体矿产**进行整理与研究。
### 采矿的本质
采矿工程并非纯粹的自然科学，其本质是一门**应用学科**或**工程技术学科**。
采矿工程的核心哲学本质是 **"妥协"**——采矿工程师不是纯粹的科学理论应用者，而是一个**平衡者**。采矿工程师的工作是在以下相互冲突的目标约束之间，寻找一个 **"相对可行的解"**，而非理论上的 **"最优解"**：
- 🛡️ 安全
- 🌱 环保
- 💰 经济
- ⚡ 效率
- ⚖️ 法律

> 对于采矿工程而言，很多情况下**最经济不一定最合理**。

### 采矿设计与实践

在采矿工程高阶段设计中，自然科学为设计提供理论依据，但其作用仅限于**原则性指导**。工程实践具有固有的**离散性**与**最小颗粒度**。例如：

- 掘进爆破的进尺（如 0.8m/循环）在实际作业中仅能以每个班次为周期进行校验；
- 巷道转弯处的几何形态亦无法实现数学意义上的理想光滑圆弧。
### 最后
> 💬 *"采矿的尽头是人情世故"*——这是一句调侃，但不可否认的是，这句话具备一定的前瞻性。
